# KITTI dataset Test
Real-time object detection and registration test based on KITTI dataset.

**[Dataset Link](https://s3.eu-central-1.amazonaws.com/avg-kitti/raw_data/2011_09_26_drive_0005/2011_09_26_drive_0005_sync.zip)**

**TO RUN:**
```
  mkdir build
  cd build
  cmake ..
  make
  cd ../bin
  ./KittiTest
```
